# Mini Task Manager

A simple CRUD Task Manager built with **Python (Flask)**, featuring a minimal **HTML/CSS/JavaScript UI** and **pytest unit tests**.  
Tasks are stored using **in-memory storage** (no database).

---

## Features

- Create tasks
- List tasks
- Filter by status
- Update title, description, and status
- Delete tasks
- Input validation & error handling
- Unit tests

---

## Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Testing:** pytest  
- **Storage:** In-memory dictionary

## Instructions to run app & tests
- git clone https://github.com/Muthuram-dev/Gold-coast-IT-solutions
- Make sure to have **pip** installed in your device and run the following command
- **pip install -r requirements.txt**
- **python app.py**
